# Bikes for Refugees Scotland

## Purpose
This is a simple landing page for Bikes for Refugees Scotland, a volunteer-led initiative that supports refugees and asylum seekers by providing donated bicycles and accessories. The page highlights the mission, upcoming events, and a Facebook events link.

## Approach
- Built with plain HTML and CSS.
- Kept the layout mobile-friendly using responsive styles and a simple two-column structure for desktop.
- Used CSS custom properties for consistent theming across the site.
- Added an inline SVG Facebook icon for the sidebar events link to avoid external font/icon dependencies.
- Centered the navigation bar in the header using CSS grid and flexbox.

## Challenges
- Ensuring the header layout stayed balanced while centering the navigation menu required switching the header container to a grid layout.
- Removing comments from the CSS while preserving readability and styling structure.
- Replacing the Facebook icon with an SVG icon that matched the page design and behaved consistently across browsers.
